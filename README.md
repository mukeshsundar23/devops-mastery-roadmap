# DevOps Mastery Roadmap

An interactive 8-week curriculum for DevOps engineering, featuring a React frontend, local state storage in PostgreSQL, and full Dockerization.

## 🚀 Quick Start

Ensure you have **Docker Desktop** or **OrbStack** running.

```bash
docker compose up --build
```

- **Frontend**: [http://localhost](http://localhost)
- **Backend API**: [http://localhost:3001](http://localhost:3001)

## 🏗️ Project Architecture

- **Frontend**: React (Vite) + Nginx
- **Backend API**: Node.js (Express)
- **Database**: PostgreSQL 15
- **Orchestration**: Docker Compose

## 🗓️ Roadmap Content
The curriculum covers 48 projects across:
- Week 1: Linux & Bash
- Week 2: Docker
- Week 3: Kubernetes
- Week 4: CI/CD
- Week 5: Terraform
- Week 6: Ansible
- Week 7: Monitoring & Observability
- Week 8: Security & Capstone Projects
